# Elfos-chkdsk
Chkdsk is a program that displays various statistics regarding an 1802 Elf-OS disk
Sample output for a two disk system is shown below
chkdsk
Disk 0:
Type 1 filesystem
Sector 0 checksum: BA93
Source disk is 240 MB. Now scanning AUs ...
1 MB is in use.
Total AUs: 61440
Free  AUs: 61207

Disk 1:
Type 1 filesystem
Sector 0 checksum: BA93
Source disk is 240 MB. Now scanning AUs ...
2 MB is in use.
Total AUs: 61440
Free  AUs: 61177
